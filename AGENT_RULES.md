# AI AGENT DEVELOPMENT RULES

This project is developed with the help of AI coding agents.

All agents working on this repository must follow these rules.

## GitHub synchronization rule

Before finishing any implementation:

1. Stage all changes
git add .

2. Create a descriptive commit
git commit -m "update: feature implementation"

3. Push changes to GitHub
git push origin main

4. Verify repository state
git status

The repository must remain clean (no pending changes).

GitHub is the single source of truth.

This ensures development can continue from:
- VS Code
- Antigravity
- Gemini
- Claude
- any development machine.

## Development rules

Do not remove working features.
Do not break the main order flow.
Always run tests before committing.
