# 🚀 Plano de Lançamento: X-Açaí SaaS

Este plano detalha os passos para sair do código local para o primeiro cliente pagante em 7 dias.

## Dia 1: Infraestrutura de Produção
- Configurar conta no Railway (Backend/Redis).
- Configurar projeto no Neon.tech (Postgres).
- Configurar projeto no Vercel (Frontend).
- Adquirir domínio (ex: `suaacai.com.br`) e configurar Cloudflare.

## Dia 2: Branding & Demo
- Customizar o Tenant `demo` com fotos profissionais de açaí.
- Configurar a Evolution API (WhatsApp) para notificações automáticas.
- Validar o fluxo de PIX com o Mercado Pago em modo produção (Sandbox primeiro).

## Dia 3: Campanha Anti-iFood (Migração)
- Imprimir 500 panfletos/adesivos com o QR Code de migração gerado pelo sistema.
- Configurar cupom de "Primeira Compra Direta" no sistema.

## Dia 4-5: Prospecção Ativa (O "Pitch")
- Visitar 5 açaíterias locais.
- Mostrar a **Vitrine Demo** no tablet/celular.
- Oferecer 30 dias grátis para o primeiro parceiro.

## Dia 6: Setup do Primeiro Cliente
- Usar a rota `/criar-delivery` para registrar o cliente.
- Ajudar o dono a cadastrar os primeiros 10 itens do cardápio.
- Entregar o QR Code de mesa e de delivery.

## Dia 7: Operação & Ajustes
- Acompanhar as primeiras ordens reais.
- Coletar feedback sobre a experiência do entregador e do cliente.
